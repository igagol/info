Analiza sprzedaży mieszkań w Sacramento oraz predykcja cen 

Predykcja cen mieszkań przy użyciu modeli machine learnigowych:
1. Random Forrest
2. Linear Regression 
3. XGBoost 

Projekt wykorzystuje następujące bilbioteki: 
1. NumPy
2. pandas
3. matplotlib
4. scikit-learn
5. seaborn 
6. scipy
7. Keras
